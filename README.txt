Use "start.bat" to quick start the program and "start +.bat" if you need additional output.

Parameters:
cfg - The path to the configuration file
text - The path to the file being processed
--log - Enables the display of file contents
--genall - Generating all files - both genconf & gentext
--genconf - help="Generating configuration file
--gentext - Generating text file
--showcount - Show the number of substitutions
--showoriginal - Shows the original line
